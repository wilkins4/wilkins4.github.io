Weather App

A simple and intuitive application that provides accurate weather information for any city.

About the App

The Weather App is designed to provide users with up-to-date weather information for any city globally. It features an intuitive UI, making it easy for users to search for and view the weather details they need.

Key Features

  • Search for any city's weather data
  • Current temperature, humidity, and wind speed details
  • 5-day weather forecast
  • Interactive map display for pinpoint weather information
