Work Index Networking Lab

Notebook Format

The target format is support-style documentation. Not every note is there yet, but this is the shape each page is moving toward.

  • Objective: What skill or problem is being tested?
  • Environment: OS, tools, services, and constraints.
  • Procedure: Repeatable commands or steps.
  • Validation: How I know the result is correct.
  • Notes: Sanitized output, screenshot, summary, or gap to fill.
  • Next step: What would make the lab stronger.
